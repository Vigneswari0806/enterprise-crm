# Enterprise CRM System

A full-stack CRM for managing leads, customers, and sales pipelines.

## Tech Stack
- Frontend: React.js + Vite + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB
- Auth: JWT + bcryptjs

## Features
- Lead tracking and deal stages
- Kanban pipeline board
- Sales performance dashboard with charts
- Email and activity logs
- Role-based access control (Admin/Manager/Sales)

## Setup
### Backend
````bash
cd server
npm install
npm run dev
````
### Frontend
````bash
cd client
npm install
npm run dev
```
